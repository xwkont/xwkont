## Scope Note

This crosswalk compares how the checked source ontologies model non-physical bearer-level entities that are not physical objects or amounts of matter: DOLCE's base-module `non-physical-endurant` / `non-physical-object` branch, plus the closest non-physical or object-adjacent analogs found in the other seven source ontologies.

DOLCE-Lite.owl is the only source that directly reifies the in-scope axis as a formal class in its core module. Its base file defines `non-physical-endurant` and `non-physical-object` directly, but not the classic DOLCE-family `Mental-Object` / `Social-Object` leaf split. That social-object apparatus lives in other DOLCE-family artifacts already crosswalked elsewhere, so this crosswalk cites those crosswalks by ID where relevant instead of re-deriving them.
UFO is a second, independently-confirmed source (found during review, not in the original draft): its 2021 paper uses the identical phrase "non-physical endurants," with a worked example, in a footnote on the constitution relation — not a reified class or formal definition, but a direct terminological and conceptual match, not incidental word use.

Core.ttl placement is still an open review question for the eventual XwkOnt concept class.
