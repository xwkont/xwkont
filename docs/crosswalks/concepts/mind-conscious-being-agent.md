## Scope Note

This draft keeps the bearer-level entity layer for agentive or conscious beings
and explicitly off-ramps the two main false positives from the raw inventory:

- GUM's mental-state terms (`Believe`, `Cognition`, `Know`, `Perception`,
  `Think`, `Wanting`, `MentalActive`, `MentalInactive`, `Disliking`,
  `Fearing`, `Liking`, `ReactionAndEmotion`, `Striving`) are modeled as mental
  processes / reactions, not as bearer classes. They belong with
  `process.md` / `event.md`, not here.
- GUM's `Female` and `Male` are pronominalization refinements under `Person`,
  not core mind/agent classes. `Person` is kept as a peripheral conscious-being
  subtype, but gender-marking subclasses are not central to the concept.
- YAMATO's `mind` is a semi-abstract category in the figure, not a continuant
  or object. It is retained as a source-side label for audit purposes, but it is
  not a core bearer-class correspondence.

The surviving standalone scope is therefore narrower than the raw inventory
suggested: conscious-being / agent as an entity class, not mental processing,
not socially conferred role, and not object-level body/organism treatment.
