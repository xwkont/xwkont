# XwkOnt

> **The Crosswalk of Foundational Ontologies**

XwkOnt is an open reference project whose sole purpose is to build a **concept-centric crosswalk** across eight foundational (upper) ontologies.

> **XwkOnt is not another foundational ontology.**
>
> It documents, compares, and connects existing foundational ontologies.

## Source ontologies in scope

BFO, DOLCE, SUMO, UFO, GFO, YAMATO, TUpper, and GUM ([ADR-0015](adr/ADR-0015-expand-source-ontology-scope-to-eight.md)).

## Persistent identifiers

- Project base: [https://w3id.org/xwkont/](https://w3id.org/xwkont/)
- Core namespace: [https://w3id.org/xwkont/core#](https://w3id.org/xwkont/core#)
- Machine-readable Turtle: request `Accept: text/turtle` against `https://w3id.org/xwkont/core`, or open [`data/ontology/core.ttl`](https://github.com/xwkont/xwkont/blob/main/data/ontology/core.ttl) on GitHub
