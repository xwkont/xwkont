# Contributing to XwkOnt

<!-- updated at: 2026-07-01 14:40 Z   (2026-07-01 10:40 EDT) -->

Thank you for your interest in XwkOnt.

This file is a short pointer for GitHub's contribution UI. The full contribution workflow — before-you-start reading list, contribution categories, review checklist, and required validation by change type — lives at:

**[`docs/governance/contributing.md`](docs/governance/contributing.md)**

Before opening a pull request, please also read:

- [`README.md`](README.md) for project scope
- [`docs/FOUNDING_PRINCIPLES.md`](docs/FOUNDING_PRINCIPLES.md) and [`docs/ARCHITECTURAL_PRINCIPLES.md`](docs/ARCHITECTURAL_PRINCIPLES.md)
- [`docs/EDITORIAL_POLICY.md`](docs/EDITORIAL_POLICY.md) for citation, source, and copyright expectations

XwkOnt documents, compares, and connects existing foundational ontologies. It does not create a new foundational ontology, reconcile philosophical disagreements between source ontologies, or introduce new domain concepts without cited evidence.
